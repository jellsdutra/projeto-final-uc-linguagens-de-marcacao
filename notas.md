# projeto-final-uc-linguagens-de-marcacao

***ENVIAR ALTERAÇÕES DO PROJETO PARA O GITHUB***

CLS -> Limpar terminal

```bash

# Verificar arquivos que não foram adicionados no GIT

git status
# Se os arquivos aparecerem em vermelho indica que não foram enviados para o GiT nem para o GitHUB
# Se os arquivos aparecerem na cor verde indica que foram enviados para o GIT HUB mas fora enviados para o GIT.

# Adicionar todos os arquivos no GIT
git add .


#criar um comentário para as alterações que serão enviadas
git commit -m "Envio dos arquivos index"

# Enviar alterações para o GIT HUB
git push origin mains
```
